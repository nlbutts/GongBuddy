# GongBuddy

LoRa radio transmitter with accel/gyro to detect impacts on a gong

DIO5 = ModeReady
DIO4 = CadDetected
DIO3 = CadDone
DIO2 = FhssChangeChannel
DIO1 = RxTimeout
DIO0 = RxDone

Mapping of DIO to the LEDs
DIO3 = CadDone
DIO4 = CadDetected
DIO0 = RxDone
DIO1 = RxTimeout
DIO2 = FhssChangeChannel
DIO5 = ModeReady
